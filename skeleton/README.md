# Your Project

A project built with the PHP Framework.

## Installation

1. Install dependencies:
   ```bash
   composer install
   ```

2. Copy environment file:
   ```bash
   cp .env.example .env
   ```

3. Create database:
   ```bash
   touch database.sqlite
   ```

4. Run migrations:
   ```bash
   php mini migrate
   ```

5. Start development server:
   ```bash
   php mini serve
   ```
